# Privacy Policy — Etsy Tag Studio

**Effective date:** May 16, 2026

This Privacy Policy describes how the **Etsy Tag Studio** Chrome extension ("the extension") handles information.

## What we collect

**Nothing.**

Etsy Tag Studio does not collect, transmit, store, or sell any personal information, browsing history, listing data, user activity, account information, or any other data about you or the listings you research.

## How the extension works

When you click the extension's toolbar icon while viewing an Etsy listing page, Etsy Tag Studio reads the public content of that single page locally in your browser. It extracts publicly-visible listing metadata (the seller's tags when exposed in the page, keywords from the listing title, and category breadcrumb) and displays them in the extension popup.

All processing happens on your computer. The extension does not send any data to us, to Etsy, or to any third party. When you close the popup, the extracted data is discarded from memory.

## Permissions explained

The extension requests the following Chrome permissions, each for a single specific purpose:

- **activeTab** — to read the Etsy listing tab you're currently viewing when you click the toolbar icon
- **storage** — reserved for future user preference storage (e.g. a saved tag library in a planned Pro tier). Not actively used in v0.1.0
- **scripting** — to inject the content script into the active Etsy tab when the popup opens, in case the script isn't already loaded
- **host permission for `*://*.etsy.com/*`** — to read Etsy listing pages. The extension never accesses any other website

## Third parties

The extension communicates with no third-party servers. It has no analytics, no telemetry, no advertising, and no integrations with external services.

## Cookies

The extension does not use cookies.

## Children's privacy

The extension is not directed at children under 13 and does not knowingly collect information from anyone, regardless of age.

## Changes to this policy

If this policy changes, the **Effective date** above will be updated and a new version of the extension will be published. There is no notification mechanism because the extension has no way to contact users — we have no email, identifier, or persistent record of who uses it.
